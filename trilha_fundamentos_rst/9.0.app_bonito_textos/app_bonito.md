## Basic font properties

* font-family
* font-weight
* font-style
* font-size
---------------------------------------------
### Font Family

* Tipo de fonte de um elemento
* Lista de fontes de um elemento
* inclui *fallback* font

```css
p {
  font-family: 'Times New Roman', times, serif;
}
```
- serif
- sans
--------------------------------------------------
## Font weight
* Peso da fonte
```css
p {
  font-weight: bold;
}
```

----------------------------------------------------
## Font style
* O estilo da fonte
```css
span {
  font-style: italic;
}
```

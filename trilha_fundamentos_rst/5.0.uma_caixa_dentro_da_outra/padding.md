## padding

Preenchimento interno da caixa

- padding-top | padding-bottom | padding-left | padding-right
- values: `length` | `porcentage` | auto

```css
div {
  /* shorthand */
  padding: 16px 12px 10px 4px;
  padding: 16px 12px 0;
  padding: 8px 16px;
  padding: 8px;
}
```
* Padding poderá causar diferença na largura de um elemento
